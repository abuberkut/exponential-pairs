Данная программа находит оптимальную последовательность 
	A и B процессов из N элементов 
	(,где Teta(A,B) будет минимальной).
То есть надо перебрать 2^N вариантов 
	и найти оптимальную последовательность
Чтобы ознакомиться более подробно можете почитать
	"Van der Corput`s method" в википедии, 
	пункт "exponent pairs"
